# README

Name
====
book Notes

Overview
読んだ本のアウトプットを管理できるアプリ

## Description
・ユーザーごとに読んだ本の管理ができる。
・読んだ本についてのノート（要約、感想、アクションプラン）を作成し、管理できる。
・他のユーザーが作成したノートを閲覧でき、いいねやコメントができる
